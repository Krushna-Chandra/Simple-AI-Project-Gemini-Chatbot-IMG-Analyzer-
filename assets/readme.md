# 📂 Project Files

This repository contains two standalone AI applications. Each file can be executed independently using Visual Studio Code.

---

## 💬 `chatbot.py`

This file contains the **Gemini AI Chatbot** implementation.

### ▶️ Run the Chatbot

Open the VS Code terminal and execute:

```bash
python chatbot.py
```

---

## 🖼️ `image.py`

This file contains the **Gemini AI Image Analyzer** built with Streamlit.

### ▶️ Run the Image Analyzer

Open the VS Code terminal and execute:

```bash
streamlit run image.py
```

---

## 📌 Prerequisites

Before running either application, make sure you have:

- Python installed
- Required dependencies installed (`pip install -r requirements.txt`)
- A valid Google Gemini API key configured in your `.env` file

---

> **Note:** These are standalone source files intended to be run locally in **Visual Studio Code**.
