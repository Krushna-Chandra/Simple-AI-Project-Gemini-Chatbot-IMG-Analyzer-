# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Krushna-Chandra/Simple-AI-Project-Gemini-Chatbot-IMG-Analyzer-.git
```

```bash
cd Simple-AI-Project-Gemini-Chatbot-IMG-Analyzer-
```

---

### Create Virtual Environment

```bash
python -m venv .venv
```

Windows

```bash
.venv\Scripts\activate
```

Linux / macOS

```bash
source .venv/bin/activate
```

---

### Install Requirements

```bash
pip install -r requirements.txt
```

---

### Create .env

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

---

### Run

```bash
streamlit run app.py
```

Visit

```
http://localhost:8501
```
