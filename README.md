# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Krushna-Chandra/Simple-AI-Project-Gemini-Chatbot-IMG-Analyzer-.git
```

Move into project

```bash
cd Simple-AI-Project-Gemini-Chatbot-IMG-Analyzer-
```

Create Virtual Environment

```bash
python -m venv .venv
```

Activate Environment

Windows

```bash
.venv\Scripts\activate
```

Linux / Mac

```bash
source .venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```
